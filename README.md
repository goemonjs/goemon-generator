# Yoemana generator for Goemon framework

## Enviroment
Node.js v10.x.x.

## Quickstart

### INSTALL
```
npm install -g yo
npm install -g https://github.com/goemonjs/goemon-generator.git
```
### GENERATE
```
yo goemon
```
## Development

### How to build
```
npm install
git submodule init
git submodule update
npm run build
```
